# BlogList

## Boot Backend

Create `.env` file in `bloglist-backend`

```
PORT=3003
MONGODB_URL=<mongodb_url>
SECRET=<secret>
```

Open command line in `bloglist-backend`

Run

```bash
npm install && npm start
```

## Boot Frontend

Open command line in `bloglist-frontend`

Run

```bash
npm install && npm start
```
